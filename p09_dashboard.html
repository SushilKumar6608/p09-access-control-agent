🔐 P09 — Access Control Anomaly Detection Agent

Hybrid Neural TPP + GraphSAGE GNN system for real-time access control anomaly detection. Dual-model fusion achieves AUC-ROC 0.9831, routed through a LangGraph 4-agent pipeline that generates plain-language security alerts via Claude Sonnet using AnoCoT prompting. Extends the author's MSc thesis TPP work (ASSA ABLOY, 2026) to the access control domain.

Design principle: The TPP and GNN detect fundamentally different anomaly types. The TPP detects temporal anomalies — unusual inter-arrival times and unexpected event sequences. The GNN detects relational anomalies — novel user-door connections and role violations. Their fusion outperforms either model alone, confirming the complementarity hypothesis.

AND fusion (both models must agree) achieves 81.6% precision with only 3.6% false positive rate — suitable for high-stakes security alerts where false alarms cause alert fatigue.
OR fusion achieves 100% recall — catches every anomaly. Use for audit logging where missing nothing matters more than false alarms.
Weight selection: Grid search over α ∈ {0.3, 0.4, ..., 0.9} found α=0.7 (TPP) + α=0.3 (GNN) maximises AUC-ROC. TPP receives higher weight as the stronger individual detector.
LangGraph 4-Agent Pipeline
🔍
Monitor Agent
Evaluates fused anomaly score. Classifies anomaly type using rule-based pre-classifier (after-hours, brute force, privilege escalation, tailgating, novel pattern).
📋
Investigation Agent
Queries HistoryStore for user access patterns, door statistics, and user-door pair frequency. Builds natural language context summary for the Alert Agent.
🤖
Alert Agent
Calls Claude Sonnet with AnoCoT prompting (LLMAD framework). Generates structured alert: what happened, why suspicious, risk assessment, recommended action.
🚨
Escalation Agent
Classifies severity (LOW / MEDIUM / HIGH / CRITICAL) using anomaly type × zone × confidence matrix. Routes HIGH/CRITICAL to human security team.
AnoCoT prompting: Implements the LLMAD framework (Liu et al., KDD 2025) — Anomaly Chain-of-Thought reasoning that structures LLM output as: (1) what happened, (2) why it is suspicious, (3) risk assessment, (4) recommended action. No fine-tuning required — few-shot in-context learning only.
Anomaly Types Detected
After-Hours Access
Access outside normal operating window (before 07:00 or after 19:00). Both TPP (unusual timing) and GNN (off-hours edge feature) contribute.
HIGH severity
Brute Force Attempt
Multiple rapid failed access attempts followed by success or alarm. TPP flags the unusual inter-arrival times between repeated events.
CRITICAL severity
Privilege Escalation
Contractor or employee accessing restricted zone (server room, roof). GNN detects the novel user-door connection with rarity=1.0.
HIGH severity
Tailgating
Same badge used twice within 5 seconds — classic piggybacking pattern. TPP detects the near-zero inter-arrival time as a strong timing anomaly.
MEDIUM severity
Door Forced Open
Door opened without a preceding badge event. Triggers system alarm. Detected by rule-based monitor and GNN structural signal.
CRITICAL severity
Novel Access Pattern
User accessing a door they have never accessed before. Pure GNN signal — edge rarity=1.0, zero historical pair frequency in training data.
MEDIUM severity

Key Findings
Finding 1 — Fusion outperforms both individual models: The fused model (AUC-ROC 0.9831) outperforms the TPP (0.9772) and GNN (0.9541) individually when evaluated at sequence level. This confirms the complementarity hypothesis — temporal and relational anomaly signals are genuinely independent and additive.
Finding 2 — GNN sequence aggregation is critical: GNN AUC-ROC improves from 0.8944 (event level) to 0.9541 (sequence level) when max-aggregating event scores per session. The most anomalous event in a session carries the diagnostic signal — averaging would dilute it.
Finding 3 — TPP epoch sensitivity: The model plateaued from epochs 5–15 then improved sharply after epoch 20. Early stopping at epoch 11 gave AUC-ROC 0.9166; full 40 epochs gave 0.9732. The loss landscape has a shallow plateau followed by a steep descent — consistent with NLL objectives on event data.
Finding 4 — GNN sweet spot at 30 epochs: AUC-ROC peaks at 30 epochs (0.8944) then degrades — classic autoencoder overfitting where the model memorises normal patterns so precisely that the threshold becomes too tight to generalise. 30 epochs is the validated optimum.
